Projets récents et services

Memory

Je vous présente un projet récent, un jeu de Memory développé à l'IUT en collaboration avec un camarade de classe. L'ensemble du code est disponible sur GitHub, où vous pouvez l'explorer.

Ce jeu se déroule dans un environnement terminal, avec un affichage conçu en ASCII, offrant une expérience ludique et rétro.

L'objectif du jeu est de former le maximum de paires. Chaque paire est constituée de cartes affichant des traductions français/anglais (par exemple, la carte "Hello" correspond à la carte "Bonjour").

Adoobe Transporrt

Dans le cadre de notre deuxième projet, nous avons collaboré à la création d'un site web statique appelé "Adoobe Transporrt". Notre équipe de trois personnes devait s'engager dans une démarche RSE (Responsabilité Sociétale des Entreprises) en proposant une alternative écologique de déplacement pour les employés d'une entreprise du numérique (Adobe dans notre cas).

Le site avait pour consigne principale d'inclure un formulaire, servant à faciliter la gestion des déplacements durables au sein de l'entreprise.
